# 119. Pascal's Triangle II

| Difficulty | Easy |
| - | - |

Given an integer `rowIndex`, return the rowIndex<sup>th</sup> **(0-indexed)** row of the **Pascal's triangle**.

### Example 1:
```
Input: rowIndex = 3
Output: [1,3,3,1]
```
### Example 2:
```
Input: rowIndex = 0
Output: [1]
```
### Example 3:
```
Input: rowIndex = 1
Output: [1,1]
``` 

### Constraints:

- `0 <= rowIndex <= 33`
